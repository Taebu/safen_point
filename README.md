# Safen_cdr
safen_cdr을 종전 trigger 방식의 자바 프로그래밍 방식으로 변경 합니다.

## Java 수정 후 구동 되게 만들기
- 해당 경로 이동
> $cd /home/point/safen_cdr

- 구동 확인 숫자가 나오면 멈추고 빌드 해야 한다.
> $sh ./status.sh

- 멈추는 명령 
> $sh ./stop.sh

- java to class
> $sh ./mask.sh

- safen_cdr 재구동
> $sh ./startw.sh

- safen_cdr 재구동 확인
> $sh ./status.sh
